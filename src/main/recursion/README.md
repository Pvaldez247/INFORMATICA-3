## EN — Summary
Console demo of classic recursive algorithms.

**Operations**
- `factorial(n)` — `n!` (n ≥ 0)
- `fibonacci(n)` — memoized recursion O(n)
- `power(base, exp)` — exponentiation by squaring (exp ≥ 0)
- `reverse(s)` — string reversal
- `isPalindrome(s)` — exact palindrome (case sensitive)
- `sumArray(int[])` — recursive sum
- `binarySearch(int[], key)` — recursive; requires sorted ascending array
- `gcd(a,b)` — Euclid’s algorithm
- `toBinaryString(n)` — builds binary string without library
- `hanoi(n, A, B, C)` — Towers of Hanoi (list of moves)

**Packages & classes**
- `recursion.app`: `RecursionUtils`
- `recursion.impl`: `MainRecursion` (console menu)
